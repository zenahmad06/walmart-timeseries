  • Open merupakan bilangan biner 1 untuk buka dan 10 untuk tidak
  • Promo juga bilangan biner 1 untuk melakukan promo dan 9 tidak
  • StateHoliday adalah hari libur. Normally all stores, with few exceptions, are closed on state holidays. Note that all schools are closed on public holidays and weekends. a = public holiday, b = Easter holiday, c = Christmas, 0 = None, karena 0 disitu ada 2 maka kita lakukan format data agar sesuai
  • school day mengindikasikan jika apakah sekolah tutup/tidak krn libur dan pengaruhnya ke toko
  • day of week merupakan nama hari

def liburan (a):
    if a =='a':
        return 'a'
    elif a=='b':
        return 'b'
    elif a=='c':
        return 'c'
    elif a=='0':
        return 'sc'
